# Code Editor
A cross-platform code editor for C/C++, Python and Java built with python. This code editor has an in-built command line for accessing the following features
### Features
- Auto-Complete
- Command Line
- Run Console
- Find and Replace
- Undo and Redo
- Color Scheme
- Custom Settings
